# 🤖 AI Personal Assistant with Text Summarizer

An AI-powered personal assistant that helps users perform smart tasks like answering queries, managing activities, and summarizing long text into concise insights using advanced language models.

---

## 🌟 Features

* 🧠 AI-powered intelligent assistant
* 💬 Chat-based interaction system
* 📄 **Text Summarizer** (convert long text into short summaries)
* ⚡ Fast and accurate responses
* 🔌 API integration with LLM models (Groq API)

---

## 🧾 Text Summarizer Module

The Text Summarizer allows users to:

* Summarize long paragraphs into short, meaningful content
* Extract key points quickly
* Improve productivity for reading articles, notes, or documents

👉 Example:

**Input:**
"Artificial Intelligence is transforming industries by automating tasks, improving efficiency, and enabling data-driven decisions..."

**Output:**
"AI improves efficiency by automating tasks and enabling better decisions."
